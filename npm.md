1,项目运行版本
npm --version (5.6.0)

2,升到最新版本
sodu npm install npm@latest -g

3,常用命令
npm help # 查看某条命令的详细帮助
npm install # 安装模块
npm uninstall # 卸载模块
npm update # 更新模块
npm start # 启动模块
npm stop # 停止模块
npm restart # 重新启动模块
npm test # 测试模块
npm version # 查看模块版本

4,更新版本
最新版本：sudo npm -g install npm@latest
